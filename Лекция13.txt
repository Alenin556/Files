Наследование конструкторов в родительском классе

Если в родительском классе указан конструктор с параметрами , необходимо указать так же конструктор с именем класса в классе наследнике с комнадой 
super 
c помощью этой команды мы даем понять java какие поля надо использовать при создании объекта наследника , и указываем в нем то что мы берем из родительского класса.

При этом при создании конструктора в классе наследнике констурктор должен быть праметризированным , с указанием в нем полей родителя и своих полей :

public Book ( int id, String name ,int price ,String author ,int pages , int publishedYear){
super (id ,name ,price)
this.author = author;
this.pages = pages;
this.publishedYear = publishedYear;
}

При создании объектов, мы можем указывать сначала родителя, а затем указать наследника чьи методы мы будем использовать.
Будет использоваться именно тот метод, который был указан после new.
Для использования определенного метода определенного наследника, нам нужно указать имя наследника в котором лежит нужный нам метод.

        prod.use();
