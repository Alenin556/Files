Лекция Циклы

Цикл for 

Цикл for состоит из трех частей:
(секция инициализации;секция условия;секция изменения)
секция инициализации - здесь мы указываем начала счетчика например int i =0; счетчик начнет работать от нуля
секция условия - здесь мы указываем какое условие необходимо будет постоянно проверять перед выполнением тела цикла, например i < 10
секция изменения - то как будет изменяться счетчик после каждого цикла(итерации)
Итого:
for ( int i = 0; i < 10; i++)
*счетчик от нуля , пока число меньше 10(и условие true) выполняем тело цикла, с шагом изменения +1.

Существует команда внутри цикла
continue
при помощи которой цикл с указанной командой завершается, и переходит к следующей итерацией, и итерация с командой continue пропускается
например 
if (i == 3){
continue
}
*по достижению счетчика цифры 3 , цифра не будет выведена в консоль.

Другой важной командой по управлению циклами является команда 
break
по достижению которой цикл завершается
например
if (i == 6){
break;
}
*по достижению счетчика цифры 6,вывод в консоль цифр завершится на 5 цифре, так как цикл с 6 цифрой был тоже прерван.
*так же для заверщения цикла можно использовать команду return и по достижению результата с использованием команды return программа завершит свое выполнение и отдаст данные на выход
*при этом использование команд return и break совместно недопустимо.
В конце программы для того чтобы убедить программу в том что если условие никогда не будет выполнено, необходимо указать команду 
return -1;
